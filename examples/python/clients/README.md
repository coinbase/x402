# x402 Python Client Examples

Simple examples showing how to use x402 with Python.

## Setup

```bash
pip install httpx requests python-dotenv eth-account web3
cp .env-example .env
# Edit .env with your EVM_PRIVATE_KEY
```

## Run

```bash
python example.py
```

The example shows both httpx (async) and requests (sync) usage. Edit the file to switch between them.
