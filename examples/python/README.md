# x402 Python Examples

Simple examples for the x402 Python SDK.

## Quick Start

```bash
cd clients
pip install httpx requests python-dotenv eth-account web3
cp .env-example .env
# Edit .env with your EVM_PRIVATE_KEY
python example.py
```
