# @x402/bazaar

x402 Payment Protocol Discovery Extension

## Installation

```bash
npm install @x402/discovery
```

## Usage

```typescript
import { /* Discovery extension exports */ } from '@x402/discovery';
```

## Development

### Build

```bash
npm run build
```

### Test

```bash
npm run test
```

### Lint

```bash
npm run lint
```

### Format

```bash
npm run format
```
