# @x402/evm

x402 Payment Protocol EVM Implementation

## Installation

```bash
npm install @x402/evm
```

## Usage

```typescript
import { /* EVM implementation exports */ } from '@x402/evm';
```

## Development

### Build

```bash
npm run build
```

### Test

```bash
npm run test
```

### Lint

```bash
npm run lint
```

### Format

```bash
npm run format
```
