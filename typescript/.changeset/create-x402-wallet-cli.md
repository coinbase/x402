Add create-x402-wallet CLI tool for zero-config agent wallet creation. Addresses issue #1759 about x402 onboarding complexity by providing a simple "install and go" experience for AI agents. Features secure key generation, multi-network support (Base Mainnet/Sepolia), JSON and .env output formats, configurable spending limits, and comprehensive documentation. Enables "npx create-x402-wallet" one-command wallet setup to reduce friction compared to MPP/Tempo alternatives.
