Add agentwallet-sdk to ecosystem directory

Adds agentwallet-sdk, a non-custodial x402 client library for AI agents featuring drop-in createX402Fetch() wrapper, on-chain budget controls, USDC payments on 17 chains via CCTP V2, and ERC-6551 token-bound accounts.
