# x402 Troubleshooting Guides

This directory contains troubleshooting guides for common x402 integration issues.

## Available Guides

### [Bazaar Discovery Issues](./bazaar-discovery-issues.md)
Diagnose and resolve issues with endpoints not appearing in facilitator discovery catalogs despite proper configuration.

**Common symptoms:**
- Endpoints have successful payments but don't appear in discovery
- Proper bazaar configuration but zero discovery results  
- Discovery document accessible but still not indexed
