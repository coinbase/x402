Add comprehensive Polygon mainnet usage examples for x402 Python SDK

This change adds three detailed Python examples demonstrating how to use x402 with Polygon mainnet:

1. **basic_payment.py** - Complete payment flow with error handling and balance checks
2. **check_balance.py** - MATIC and USDC balance verification with gas cost estimation  
3. **compare_networks.py** - Side-by-side comparison of Polygon vs Base costs and features

Key features:
- Real-world cost comparisons showing 85% savings on Polygon vs Base
- Detailed gas cost estimation and transaction capacity analysis
- Comprehensive setup instructions with multiple RPC endpoint options
- Balance checking with funding recommendations
- Network selection guidance based on use case requirements

The examples complement the recent addition of Polygon mainnet support (#1791) by providing practical implementation guidance for developers choosing between networks for x402 micropayments.
